# College Reading Material Generator

This Streamlit application generates comprehensive reading materials for college students using AI. It processes course outlines and creates detailed chapter content with proper formatting.

## Features

- Upload course outline in DOCX format
- Process course outline with Gemini API to extract chapters and topics
- Generate detailed reading materials using Deepseek Reasoner API
- Download generated materials as properly formatted DOCX files
- Support for bold and italic formatting in generated content
- Justified paragraphs with proper indentation and spacing

## How to Use

1. Upload your course outline in DOCX format
2. Process the outline with Gemini API to extract chapters and topics
3. Select which chapters you want to generate reading materials for
4. Generate the materials using Deepseek API
5. Download the generated materials as DOCX files

## Requirements

- Gemini API key
- Deepseek API key

## Dependencies

- streamlit
- python-docx
- requests

## Deployment

This app is designed for deployment on Streamlit Cloud.
